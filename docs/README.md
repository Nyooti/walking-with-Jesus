# Walking with Jesus Christ - Live Website

This is the live deployment of the Walking with Jesus Christ website hosted on GitHub Pages.

## 🌐 Access the Website

**Main Website**: https://nyooti.github.io/walking-with-Jesus/

**Admin Panel**: https://nyooti.github.io/walking-with-Jesus/admin.html

## 📝 Features

- **Blog Management**: Full CRUD operations for blog posts
- **Sermons**: Browse and view sermon content
- **Missions**: Learn about mission work
- **Testimonies**: Read faith testimonies
- **Prayer**: Prayer requests and resources
- **Contact**: Get in touch with the ministry

## 🚀 Backend API

The website uses a backend API for blog management. For local development:

1. Clone the repository
2. Run `python3 backend.py` to start the API server (port 5000)
3. Run `python3 docs/server.py` to start the frontend server (port 8000)

## 📱 Responsive Design

The website is fully responsive and works on:
- Desktop computers
- Tablets
- Mobile phones

## 🙏 About

This website is dedicated to sharing Christian content, including theological articles, cultural commentary, and biblical interpretation.
